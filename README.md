# 👨‍💼 Employee Management System

A console-based Employee Management System built using Java, JDBC, and PostgreSQL.

## 🚀 Features

✅ Add Employee

✅ View Employees

✅ Update Employee

✅ Delete Employee

## 🛠️ Technologies Used

* ☕ Java
* 🔗 JDBC
* 🐘 PostgreSQL
* 📦 OOP Concepts
* 🎯 PreparedStatement
* 🖥️ Console Application

## 📂 Database Table

```sql
CREATE TABLE employee (
    emp_id INT PRIMARY KEY,
    emp_name VARCHAR(100),
    emp_salary DOUBLE PRECISION,
    department VARCHAR(100)
);
```

## ▶️ How to Run

1. Install PostgreSQL
2. Create Database
3. Create Employee Table
4. Add PostgreSQL JDBC Driver
5. Run `EmployeeManagementSystem.java`

## 📚 Concepts Practiced

* Object-Oriented Programming (OOP)
* JDBC Connectivity
* CRUD Operations
* PreparedStatement
* Database Integration
* Exception Handling

## 🎯 Learning Outcome

This project helped me learn how to connect Java applications with PostgreSQL databases and perform CRUD operations using JDBC.
